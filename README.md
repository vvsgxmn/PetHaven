## 项目介绍

本系统是一个宠物寄养家庭服务平台，为宠物主人和寄养家庭提供在线对接服务，支持寄养搜索、需求发布、订单流转、状态跟踪等完整业务闭环。

## 功能模块

### 用户端

- **寄养搜索**：多维度筛选寄养家庭，按城市、宠物类型、价格、评分等条件搜索匹配
- **寄养需求**：发布寄养需求，包含宠物信息、时间范围、特殊要求等
- **订单管理**：查看全部订单，支持待接单、已接单、服务中、已完成、已取消等状态筛选
- **宠物管理**：添加和管理自己的宠物信息
- **实时跟踪**：寄养期间实时查看宠物状态更新
- **消息通知**：接收订单状态变更、审核结果等系统通知
- **个人中心**：修改个人信息、密码等
- **寄养家庭申请**：用户可申请成为寄养家庭

### 管理后台

- **数据面板**：展示用户数、寄养家庭数、订单数、待审核等统计数据
- **用户管理**：查看所有用户，支持编辑、消息发送、冻结等操作
- **宠物管理**：管理平台所有宠物信息
- **资质审核**：审核寄养家庭申请资质
- **需求管理**：管理所有寄养需求
- **订单管理**：管理平台全部订单
- **投诉管理**：处理用户投诉

## 技术架构

| 层次 | 技术 |

| UI组件 | Bootstrap 5 + Bootstrap Icons |
| 构建工具 | Vite 7 |
| 数据库 | MySQL 8 |
| 认证 | JWT |
| 工具库 | Hutool、Lombok |
